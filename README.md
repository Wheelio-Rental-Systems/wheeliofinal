# Wheelio Project

This project is organized into two main components:

## Directory Structure

- **`frontend/`**: The React-based frontend application.
  - To run: `cd frontend` then `npm run dev`
- **`wheelio-backend/`**: The Spring Boot backend application.
  - To run: Open in IntelliJ or use Maven to run.

## Getting Started

1.  **Frontend**: Navigate to the `frontend` directory. Install dependencies with `npm install` if needed, then start the development server with `npm run dev`.
2.  **Backend**: Navigate to the `wheelio-backend` directory. Configure your database settings if necessary and start the Spring Boot application.
